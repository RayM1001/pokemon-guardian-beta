## 已知部署注意事項

1. **JSON 路徑**：fetch 使用 `data/pokemon.json`（相對路徑），在 GitHub Pages 根目錄部署正確。若使用子路徑（如 `username.github.io/repo-name/`）則相對路徑仍正確，不需調整。
2. **.nojekyll**：根目錄已建立 `.nojekyll`，避免 GitHub Pages Jekyll 處理影響底線開頭的資料夾。
3. **Canvas API**：分享圖片使用 Canvas API，需現代瀏覽器支援（iOS 12+ / Android 5+）。
4. **剪貼簿 API**：`navigator.clipboard.writeText()` 在 HTTPS 環境正常；GitHub Pages 自動提供 HTTPS。
